# Uber Rides Data Analysis 

## Project Overview
This project explores Uber ride data to uncover patterns in ride purposes, categories, distances, and time-based trends.

The analysis includes data cleaning, feature engineering, and exploratory data analysis (EDA) using Python.

This project was originally based on a guided tutorial and extended with additional analysis and improvements.

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## Dataset
The dataset contains information about Uber rides, including:
- Start Date & End Date
- Category (Business / Personal)
- Start & Stop Locations
- Distance (Miles)
- Purpose of the ride

---

## What I Did
- Cleaned and preprocessed the dataset
- Handled missing values
- Converted date columns into datetime format
- Extracted new features:
  - Hour of ride
  - Day of week
  - Month
  - Day vs Night classification
- Performed exploratory data analysis (EDA)
- Created visualizations to identify patterns

---

## Key Insights
- Business rides occur more frequently than personal rides
- Most rides happen during working hours
- Certain purposes (like meetings) dominate business trips
- Distance distribution shows most rides are short trips
- Peak ride activity varies by time of day and weekday
